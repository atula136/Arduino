public class ImapPushService
extends RunnablePushService

Registers a message handler with JavamailImapMontitor to send alerts when new messages are received. Only emails that contain the text/plain mime type are supported. If an email is text/html or something else, you will get the from/subject only.

Displays a maximum of 640 characters of an email message (approximately 8 pages on an LCD). Any content beyond 640 characters is discarded.
