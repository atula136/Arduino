Class Summary
Content Represents one or more pages of related content.

Page Represents content to be displayed on a remote LCD.
PaginationRules A collection of rules for describing how a blob of text should be formated for display on a LCD
Word A word is a contiguous sequence of non white/LF characters or a single whitespace or single LF
WordList Provides access to List of Words
