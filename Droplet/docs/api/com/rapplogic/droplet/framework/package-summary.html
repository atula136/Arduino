Package com.rapplogic.droplet.framework

Interface Summary
ServiceContext Exposes the Droplet API functionality to services, allowing services to send messages, alerts, format content etc.
 

Class Summary
Credentials Reads a file containing a username/password pair
Droplet Receives XBee RX Packets from the remote XBee/Arduino/LCD for a service and responds with a string to be displayed on the LCD.
LcdProperties Specifies the dimensions of the LCD: number of lines and line size.
Message Represents one or more pages of content to be sent to a remote XBee for display.
 

Exception Summary
DeliveryException Thrown if a message to a remote is undeliverable.
